// jquery에서 class 조작

append - 선택한 태그의 안, 마지막에 추가
prepend - 선택한 태그의 안, 처음에 추가
before - 선택한 태그의 앞쪽에 추가
after - 선택한 태그의 뒤쪽에 추가

remove - 선택한 태그를 삭제
empty - 선택한 태그 안을 비움

removeClass - 선택한 태그에 지정한 클래스를 삭제
addClass - 선택한 태그에 지정한 클래스를 추가
